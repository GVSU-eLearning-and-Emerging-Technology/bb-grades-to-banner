Instructions

Grades that have been entered or calculated in Blackboard can be brought into Banner, removing the need to retype student grades from scratch. This process involves downloading a file from both Blackboard and Banner, making sure the information in each file properly matches, and then importing the grade file back into Banner.

This tool simplifies that process by taking your two files and reconciling them for you.

Note: You will still need to take the final step of importing your reconciled file into Banner.

  1. In Blackboard, choose to download the "Full Gradebook". Under "Record Details", all you need to select is the name of the column that contains your final grade. This column is usually named "Overall Grade." (learn more about this process in the Service Portal)
  2. When exporting the "template file" from Banner, select the option for "Excel spreadsheet (.xlsx)"
  3. These files contain FERPA-protected information. Please delete each of them when you have finished submitting your grades in Banner.
